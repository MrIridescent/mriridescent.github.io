1.  **Sovereign Contracts**: Every data exchange is governed by a legally-binding, machine-readable contract. This contract specifies the data provider, consumer, and a granular usage policy (e.g., region, retention period).
2.  **Geopatriation Check**: Before any data is exchanged, the connector validates the consumer's region against the local node's region. Any attempt to export data to an unauthorized region is immediately blocked and flagged as a sovereignty violation.
