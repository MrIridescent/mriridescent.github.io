## **Core Operational Logic**
1.  **Cosmos Curator**: Automates the "DataOps" of sensor-level intelligence. It filters, annotates, and deduplicates high-bandwidth sensor data (LiDAR, Radar, Camera) to prepare it for world-state mapping.
2.  **Cosmos Predict**: Utilizes generative AI to predict the future state of the world up to 30 seconds ahead. This "imagination" allows robots to anticipate human movements or machinery failures.
3.  **Cosmos Reason**: A multimodal reasoning engine that uses **Chain-of-Thought (CoT)** to interpret space-time logic. It distinguishes between static obstacles and dynamic, high-priority risks.
